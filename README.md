Saad Admin Platform

Saad Admin Platform is a modern, fully responsive Admin Dashboard / SaaS Platform built using HTML, CSS, and Vanilla JavaScript, focused on performance, clean UI/UX, and real-world admin features.

This project demonstrates strong front-end architecture, state management, and user experience design without relying on heavy frameworks.


 Features

 Authentication & Security
 • Login & Register system
 • Protected routes (Guard system)
 • Logout functionality
 • Persistent auth state using LocalStorage

 Theme & Language System
 • Light / Dark Mode
 • Preferences saved and applied globally
 • Changes applied only after clicking Save Preferences

 Dashboard & Analytics
 • KPI cards (Users, Revenue, Growth)
 • Interactive Line Chart (hover values)
 • Bar Chart analytics
 • Fully responsive charts
 • Dark mode compatible charts

 User Management
 • Users table
 • Role selection (Admin / User / Viewer)
 • Action buttons
 • Responsive table layout
 • Dark mode table support

 Navigation & Layout
 • Collapsible sidebar with overlay
 • Mobile-friendly menu button
 • Active navigation states
 • Smooth transitions and hover effects

 Responsive Design
 • Mobile-first layout
 • Optimized for desktop, tablet, and mobile
 • Touch-friendly UI elements

 UX & UI Highlights

 Tech Stack
 • HTML5
 • CSS3 (Flexbox, Grid, Responsive Design)
 • JavaScript (Vanilla JS)
 • Chart.js
 • LocalStorage API

 No frameworks used.


 How to Run
 1. Clone or download the project
 2. Open index.html in your browser
 3. Login to access the dashboard
 4. Use Settings to switch language or theme

No backend required.
